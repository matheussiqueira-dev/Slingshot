# Slingshot

Slingshot is a strategy-first bubble shooter concept that blends webcam input with fast, tactical decisions.

## Status

This repository is being prepared for the first public drop. Code and assets will land here next.

## Vision

- Webcam-driven aim and control
- Real-time strategy hints using local heuristics (no external AI services)
- Fast, arcade-style bubble shooting with scoring combos

## Planned Stack

- React + TypeScript
- Vite
- MediaPipe Hands (camera tracking)
- Canvas rendering
